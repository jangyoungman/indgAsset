# 프론트엔드 디자인 개선 스펙

## 목표
자산관리 시스템의 프론트엔드를 SaaS 비즈니스 대시보드 스타일(Notion/Linear/Vercel)로 개선한다.

## 디자인 결정사항
- **테마**: 다크 사이드바 + 화이트 콘텐츠 (GitHub/Vercel 스타일)
- **대시보드 레이아웃**: 상단 4개 통계 카드 + 하단 최근 활동 테이블
- **기술**: Tailwind CSS 3 설치 및 설정 (기존 Tailwind 클래스 활용)

## 기술 구성
- Tailwind CSS 3 + PostCSS + Autoprefixer 설치
- tailwind.config.js, postcss.config.js 생성
- src/index.css에 Tailwind 디렉티브 추가
- 별도 UI 라이브러리 없이 Tailwind 유틸리티만 사용

## 공통 레이아웃 (Layout.jsx)
- 좌측 사이드바: w-60, bg-gray-900, 고정
  - 상단 로고/앱 이름 (인디고 포인트)
  - 네비게이션: 아이콘 + 텍스트, 활성 메뉴는 bg-indigo-500/10 + text-indigo-400
  - 하단 사용자 정보 + 로그아웃
- 우측 메인 영역: bg-gray-50, p-6
  - 상단 헤더: 페이지 타이틀 + 알림 벨 + 유저 아바타

## 로그인 페이지 (Login.jsx)
- 중앙 정렬 카드, bg-white, 그림자, 둥근 모서리
- 인디고 계열 로그인 버튼
- 깔끔한 입력 필드 (포커스 시 인디고 테두리)

## 대시보드 (Dashboard.jsx)
- 4개 통계 카드 가로 배열: 전체 자산 / 사용가능 / 사용중 / 승인대기
  - 각각 색상 구분 (그린, 인디고, 앰버 등)
- 최근 활동 테이블: 카드 형태, 액션 아이콘 + 타임스탬프
- 추가 섹션: 카테고리별/부서별 현황, 워런티 만료 알림

## 자산 목록 (AssetList.jsx)
- 상단: 검색바 + 필터 드롭다운 + 자산 추가 버튼
- 깔끔한 테이블: 호버 시 bg-gray-50, 상태 뱃지 색상 코딩
- 페이지네이션 하단 정렬

## 색상 시스템
