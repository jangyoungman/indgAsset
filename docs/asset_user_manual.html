1. 시스템 소개

INNODIGM 자산관리 시스템은 회사 자산(노트북, 모니터, 사무기기 등)의 등록, 조회, 대여/반납, 현황 관리를 위한 웹 애플리케이션입니다.

항목내용
접속 주소https://asset.indg.co.kr
지원 브라우저Chrome, Edge, Firefox (최신 버전)
모바일반응형 지원 (스마트폰/태블릿 접속 가능)

일반 사용자 자산 목록, 대여/반납 자산 조회, 본인 대여 내역 확인

4. 자산 검색 (메인 화면) 전체

로그인 후 첫 화면입니다. 자연어로 자산을 검색할 수 있습니다.

사용 방법

1
검색창에 찾고 싶은 자산을 자연어로 입력합니다.
2
Enter 키 또는 검색 버튼을 클릭합니다.
3
AI가 검색 조건을 자동으로 분석하여 결과를 보여줍니다.
Shift + Enter로 줄바꿈이 가능합니다.

자연어 검색 예시

AI가 입력 내용을 분석하여 적절한 검색 조건을 자동 생성합니다.

입력 예시검색 조건
노트북 폐기 외의 목록카테고리=노트북, 상태 제외=폐기
IT팀 빼고 사용중인 노트북카테고리=노트북, 상태=사용 중, 부서 제외=IT팀
올해 구매한 장비구매일 ≥ 2026-01-01
100만원 이상 노트북카테고리=노트북, 금액 ≥ 1,000,000원
보증 만료 임박한 장비보증만료일 조건
개발팀 모니터 사용중카테고리=모니터, 상태=사용 중, 부서=개발팀

키워드 검색

검색창 아래의 "키워드 검색" 탭을 클릭하면 카테고리, 상태, 부서, 사용자 칩을 직접 클릭하여 검색어를 조합할 수 있습니다.

검색 결과

  • 검색 후 상단에 필터 태그가 표시되어 어떤 조건으로 검색되었는지 확인 가능
  • 제외 조건은 빨간색 태그로 구분됩니다.
  • 결과 목록에서 자산을 클릭하면 상세 화면으로 이동합니다.
AI 검색 장애 시 키워드 기반 검색으로 자동 전환됩니다. 키워드 검색은 정확한 이름 매칭으로 동작하며, 부정 조건·금액·날짜 범위는 지원하지 않습니다.

5. 대시보드 매니저 이상

좌측 메뉴의 "대시보드"를 클릭하면 자산 현황을 한눈에 파악할 수 있습니다.

  • 자산 현황 — 총 자산 수, 상태별(사용 가능/사용 중/정비 중/폐기) 카운트
  • 카테고리별 분포 — 노트북, 모니터, 사무가구 등 분류별 수량
  • 최근 활동 — 최근 등록/수정/대여/반납 내역

6. 자산 목록 조회 전체

좌측 메뉴의 "자산 목록"을 클릭하면 등록된 자산을 확인할 수 있습니다.

검색 및 필터

  • 검색 — 자산명, 자산코드, 시리얼넘버 등으로 검색
  • 상태 필터 — 사용 가능 / 사용 중 / 정비 중 / 폐기
  • 카테고리 필터 — 노트북, 모니터, 사무가구 등

목록 정보

항목설명

자산명자산 이름
카테고리노트북, 모니터, 사무가구 등
상태사용 가능 / 사용 중 / 정비 중 / 폐기
사용자현재 사용(대여) 중인 직원
위치자산이 위치한 장소

7. 자산 상세 조회 전체

자산 목록에서 자산을 클릭하면 상세 정보를 확인할 수 있습니다.

  • 기본 정보 — 자산코드, 자산명, 카테고리, 제조사, 모델, 시리얼넘버
  • 취득 정보 — 구매일, 구매가, 보증만료일
  • 현재 상태 — 상태, 위치, 부서, 사용자
  • MAC 주소 — 네트워크 장비의 MAC 주소
  • 이력 — 등록/수정/대여/반납 등의 변경 이력
  • 비고 — 추가 메모 사항

9. 자산 수정 매니저 이상

1
자산 상세 화면에서 "수정" 버튼 클릭
2
정보 수정 후 "저장" 클릭

수정 내역은 자산 이력에 자동 기록됩니다.

10. 자산 일괄 등록 관리자

여러 자산을 엑셀 파일로 한번에 등록할 수 있습니다.

1
자산 목록 → "일괄 등록" 버튼 클릭
2
"템플릿 다운로드"를 클릭하여 엑셀 양식 다운로드
3
엑셀 파일에 데이터 입력 (시트: IT장비, 사무용품, 기타)
4
파일을 드래그하거나 클릭하여 업로드
5
미리보기에서 데이터 확인 후 "일괄 등록" 클릭

엑셀 시트별 용도

시트용도필수 항목
IT장비노트북, 모니터, 서버 등자산명, 카테고리
사무용품책상, 의자 등자산명, 카테고리
기타차량, 기타 자산자산명, 카테고리
주의사항:
  • 시리얼넘버가 이미 등록된 경우 중복 오류가 발생합니다.
  • 카테고리는 시스템에 등록된 이름과 정확히 일치해야 합니다.
  • 한 번에 최대 200건까지 등록 가능합니다.
  • 헤더는 반드시 한글로 입력하세요. (자산명, 카테고리, 시리얼넘버 등)

11. 대여 / 반납 전체

좌측 메뉴의 "대여/반납"을 클릭하면 3개 탭으로 구성된 화면이 나타납니다.

내 대여 현황 탭

본인의 대여 요청/진행/반납 이력을 확인할 수 있습니다.

  • 상태 필터 — 요청중 / 승인 / 사용중 / 반납완료 / 거절
  • 반납 — "사용중" 상태의 자산에서 반납 버튼 클릭 → 반납 메모 입력 → 처리

대여 요청 탭

"사용 가능" 상태인 자산 목록이 표시됩니다.

1
대여할 자산의 "대여 요청" 버튼 클릭
2
반납 예정일 선택 및 요청 메모 입력
3
"대여 요청" 버튼 클릭

요청이 접수되면 관리자에게 이메일이 발송됩니다. 이미 요청 중인 자산은 "요청중" 뱃지가 표시됩니다.

승인 대기 탭 매니저 이상

대여 요청 건을 승인 또는 거절합니다.

1
요청 건의 "승인" 또는 "거절" 버튼 클릭
2
응답 메모 입력 후 확인

승인/거절 시 요청자에게 이메일과 알림이 발송됩니다. 승인 시 자산 상태가 자동으로 "사용 중"으로 변경됩니다.

12. 시스템 관리 관리자

좌측 메뉴의 "시스템 관리"를 클릭하면 하위 메뉴가 펼쳐집니다.

사용자 관리

시스템 관리 → "사용자 관리"

  • 사용자 목록 — 전체 사용자 조회 (이름, 이메일, 역할, 부서, 상태)
  • 사용자 등록 — 이름, 이메일, 비밀번호, 역할, 부서 입력
  • 사용자 수정 — 이름, 역할, 부서, 연락처 변경
  • 역할 변경 — 관리자 / 매니저 / 일반 사용자
  • 활성화/비활성화 — 계정 사용 중지 또는 활성화
  • 계정 잠금 해제 — 로그인 실패로 잠긴 계정 해제
  • 비밀번호 초기화 — 임시 비밀번호 발급 (사용자에게 전달)
  • VPN 인증서 — VPN+ (생성) / VPN🔑 (조회/다운로드/삭제)

이메일 수신 설정 (사용자 수정 시)

항목설명
홈페이지 관리자홈페이지 관리 권한 부여
홈페이지 문의 이메일 수신홈페이지 문의 접수 시 이메일 수신
자산 대여 요청 이메일 수신자산 대여 요청 시 이메일 수신

공통코드 관리

시스템 관리 → "공통코드 관리"

자산 상태, 역할 등 시스템에서 사용하는 코드값을 관리합니다.

  • ASSET_STATUS — 자산 상태 (사용 가능, 사용 중, 정비 중, 폐기)
  • 코드 추가/수정/순서변경/활성화/비활성화 가능

부서 관리

시스템 관리 → "부서 관리"

부서 추가 및 수정을 관리합니다.

13. VPN 인증서 전체

WireGuard VPN 인증서를 다운로드하여 외부에서 사내 네트워크에 접속할 수 있습니다.

본인 VPN 인증서 다운로드

화면 상단 헤더의 VPN 다운로드 아이콘을 클릭하면 본인의 VPN 설정 파일(.conf)이 다운로드됩니다.

참고: VPN 인증서가 생성되지 않은 경우 아이콘이 표시되지 않습니다. 관리자에게 생성을 요청하세요.

VPN 설정 방법

1
WireGuard 앱 설치 (Windows / Mac / iOS / Android)
2
다운로드 받은 .conf 파일을 WireGuard 앱에서 Import
3
터널을 활성화하면 사내 네트워크에 접속됩니다

VPN 인증서 관리 관리자

시스템 관리 → 사용자 관리에서 각 사용자 옆의 VPN 버튼으로 관리합니다.

버튼동작
VPN+인증서 생성 (생성 후 WireGuard 서버에 피어 추가 필요)
VPN🔑인증서 조회, 설정파일 다운로드, 삭제

14. FAQ

Q. VPN 다운로드 아이콘이 보이지 않습니다.

관리자가 VPN 인증서를 아직 생성하지 않은 경우입니다. 관리자에게 요청하세요.

Q. 대여 요청이 안 됩니다.

"사용 가능" 상태의 자산만 대여 요청이 가능합니다. 이미 요청 중인 자산은 "요청중" 뱃지가 표시됩니다.

Q. 로그인이 안 됩니다.

이메일 주소와 비밀번호를 확인하세요. 5회 실패 시 계정이 잠기며, 관리자에게 해제를 요청해야 합니다.

Q. 비밀번호를 잊었습니다.

관리자에게 비밀번호 초기화를 요청하세요.

Q. 자산 등록 버튼이 보이지 않습니다.

관리자 또는 매니저 권한이 필요합니다. 관리자에게 권한 변경을 요청하세요.

Q. 일괄 등록 시 오류가 발생합니다.

다음 사항을 확인하세요:

  • 엑셀 시트 이름이 "IT장비", "사무용품", "기타" 중 하나인지
  • 헤더가 한글(자산명, 카테고리 등)로 되어있는지
  • 카테고리명이 시스템에 등록된 이름과 일치하는지
  • 시리얼넘버가 중복되지 않는지

Q. 자산 코드는 어떻게 부여되나요?

구매일 기준 연도 + 순번으로 자동 생성됩니다. (예: AST-2026-001, AST-2026-002)

Q. 삭제된 자산은 복구할 수 있나요?

삭제된 자산은 "폐기" 상태로 처리되며, 목록에서 필터로 확인 가능합니다.
