1. 시스템 소개

INNODIGM 자산관리 시스템은 회사 자산(노트북, 모니터, 사무기기 등)의 등록, 조회, 대여/반납, 현황 관리를 위한 웹 애플리케이션입니다.

항목내용
접속 주소https://asset.indg.co.kr
지원 브라우저Chrome, Edge, Firefox (최신 버전)
모바일반응형 지원 (스마트폰/태블릿 접속 가능)

2. 사용자 권한

역할에 따라 사용할 수 있는 기능이 다릅니다.

역할접근 가능 메뉴주요 권한
관리자 전체 메뉴 자산 등록/수정/삭제, 일괄등록, 사용자 관리, 시스템 설정
매니저 대시보드, 자산 목록, 대여/반납 자산 등록/수정, 대여/반납 처리
일반 사용자 자산 목록, 대여/반납 자산 조회, 본인 대여 내역 확인

3. 로그인

1
https://asset.indg.co.kr 접속
2
이메일과 비밀번호 입력 후 로그인

이메일은 @indg.co.kr 회사 이메일을 사용합니다.

초기 비밀번호: indg2016! — 최초 로그인 후 보안을 위해 반드시 비밀번호를 변경하세요.
로그인 실패 5회 시 계정이 잠깁니다. 잠금 해제는 관리자에게 문의하세요.

이메일만 입력하여 로그인

이메일의 @indg.co.kr 부분은 생략하고 아이디만 입력해도 됩니다.

예: gellotin@indg.co.kr → gellotin만 입력

4. 대시보드 매니저 이상

로그인 후 첫 화면입니다. 자산 현황을 한눈에 파악할 수 있습니다.

  • 자산 현황 — 총 자산 수, 상태별(사용 가능/사용 중/정비 중/폐기) 카운트
  • 카테고리별 분포 — 노트북, 모니터, 사무가구 등 분류별 수량
  • 최근 활동 — 최근 등록/수정/대여/반납 내역

5. 자산 목록 조회 전체

좌측 메뉴의 "자산 목록"을 클릭하면 등록된 자산을 확인할 수 있습니다.

검색 및 필터

  • 검색 — 자산명, 자산코드, 시리얼넘버 등으로 검색
  • 상태 필터 — 사용 가능 / 사용 중 / 정비 중 / 폐기
  • 카테고리 필터 — 노트북, 모니터, 사무가구 등

목록 정보

항목설명
자산 코드자동 생성되는 고유 코드 (예: AST-2026-001)
자산명자산 이름
카테고리노트북, 모니터, 사무가구 등
상태사용 가능 / 사용 중 / 정비 중 / 폐기
사용자현재 사용(대여) 중인 직원
위치자산이 위치한 장소

6. 자산 상세 조회 전체

자산 목록에서 자산을 클릭하면 상세 정보를 확인할 수 있습니다.

  • 기본 정보 — 자산코드, 자산명, 카테고리, 제조사, 모델, 시리얼넘버
  • 취득 정보 — 구매일, 구매가, 보증만료일
  • 현재 상태 — 상태, 위치, 부서, 사용자
  • MAC 주소 — 네트워크 장비의 MAC 주소
  • 이력 — 등록/수정/대여/반납 등의 변경 이력
  • 비고 — 추가 메모 사항

7. 자산 등록 매니저 이상

자산 목록 화면에서 "자산 등록" 버튼을 클릭합니다.

필수 입력 항목

모델모델명
구매일자산 구매 날짜

8. 자산 수정 매니저 이상

1
자산 상세 화면에서 "수정" 버튼 클릭
2
정보 수정 후 "저장" 클릭

수정 내역은 자산 이력에 자동 기록됩니다.

9. 자산 일괄 등록 관리자

여러 자산을 엑셀 파일로 한번에 등록할 수 있습니다.

1
자산 목록 → "일괄 등록" 버튼 클릭
2
"템플릿 다운로드"를 클릭하여 엑셀 양식 다운로드
3
엑셀 파일에 데이터 입력 (시트: IT장비, 사무용품, 기타)
4
파일을 드래그하거나 클릭하여 업로드
5
미리보기에서 데이터 확인 후 "일괄 등록" 클릭

엑셀 시트별 용도

시트용도필수 항목
IT장비노트북, 모니터, 서버 등자산명, 카테고리
사무용품책상, 의자 등자산명, 카테고리
기타차량, 기타 자산자산명, 카테고리
주의사항:
  • 시리얼넘버가 이미 등록된 경우 중복 오류가 발생합니다.
  • 카테고리는 시스템에 등록된 이름과 정확히 일치해야 합니다.
  • 한 번에 최대 200건까지 등록 가능합니다.
  • 헤더는 반드시 한글로 입력하세요. (자산명, 카테고리, 시리얼넘버 등)

10. 대여 / 반납 전체

좌측 메뉴의 "대여/반납"에서 자산의 대여 및 반납 이력을 관리합니다.

대여

1
자산 상세 화면에서 "대여" 버튼 클릭
2
대여자(사용자) 선택 후 확인

대여 시 자산 상태가 "사용 중"으로 자동 변경됩니다.

반납

1
자산 상세 화면에서 "반납" 버튼 클릭

반납 시 자산 상태가 "사용 가능"으로 자동 변경됩니다.

11. 사용자 관리 관리자

좌측 메뉴의 "사용자 관리"에서 시스템 사용자를 관리합니다.

  • 사용자 목록 — 전체 사용자 조회
  • 역할 변경 — 관리자 / 매니저 / 일반 사용자 권한 설정
  • 계정 잠금 해제 — 로그인 실패로 잠긴 계정 해제
  • 비밀번호 초기화 — 사용자 비밀번호 재설정

12. 시스템 관리 관리자

공통코드 관리

좌측 메뉴 → 시스템 관리 → "공통코드 관리"

자산 상태, 카테고리 등 시스템에서 사용하는 코드값을 관리합니다.

  • ASSET_STATUS — 자산 상태 (사용 가능, 사용 중, 정비 중, 폐기)
  • 코드 추가/수정/순서변경/활성화/비활성화 가능

부서 관리

좌측 메뉴 → 시스템 관리 → "부서 관리"

부서 추가, 수정, 삭제를 관리합니다.

현재 부서
IT팀, 경영지원팀, 개발팀, 마케팅팀, 인사팀

13. FAQ

Q. 로그인이 안 됩니다.

이메일 주소와 비밀번호를 확인하세요. 5회 실패 시 계정이 잠기며, 관리자에게 해제를 요청해야 합니다.

Q. 비밀번호를 잊었습니다.

관리자에게 비밀번호 초기화를 요청하세요.

Q. 자산 등록 버튼이 보이지 않습니다.

관리자 또는 매니저 권한이 필요합니다. 관리자에게 권한 변경을 요청하세요.

Q. 일괄 등록 시 오류가 발생합니다.

다음 사항을 확인하세요:

  • 엑셀 시트 이름이 "IT장비", "사무용품", "기타" 중 하나인지
  • 헤더가 한글(자산명, 카테고리 등)로 되어있는지
  • 카테고리명이 시스템에 등록된 이름과 일치하는지
  • 시리얼넘버가 중복되지 않는지

Q. 자산 코드는 어떻게 부여되나요?

구매일 기준 연도 + 순번으로 자동 생성됩니다. (예: AST-2026-001, AST-2026-002)

Q. 삭제된 자산은 복구할 수 있나요?

삭제된 자산은 "폐기" 상태로 처리되며, 목록에서 필터로 확인 가능합니다.
